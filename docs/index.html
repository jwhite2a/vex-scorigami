What is Scorigami?

Scorigami is a concept proposed by Jon Bois. Fundamentally, the chart above represents every possible win-lose score pair. If such a pair is scored, the box gets filled. You can find more information about the origin and rules of Scorigami here.

How does this work?

Thanks to the amazing API at vexdb.io, this entire project was made without major difficulty!
